## Workflow Kerja Kelompok di GitHub (Feature Branch Workflow)
1. Membuat clone repository

2. Mulai dengan address master

3. Membuat branch baru untuk setiap fitur

4. Kerjakan, add, dan commit di Branch masing-masing

5. Push branch fitur ke GitHub

6. Membuat Pull Request (PR)

7. Lakukan review, diskusi dan merge

8. Sinkronasi ulang setelah merge
